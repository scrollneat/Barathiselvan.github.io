# Day 25 – Working with CSV Data & Introduction to Pandas

## 📌 Overview
This lesson introduces how to work with **CSV (Comma Separated Values) files** in Python and explains **why Pandas is the preferred library** for data analysis. We begin with basic file handling, move to Python’s built‑in `csv` module, and then transition to **Pandas** for cleaner, faster, and more scalable data processing.

---

## 🎯 Learning Objectives
By the end of this lesson, you will be able to:
- Understand what CSV files are and where they are used
- Read CSV files using:
  - Basic file handling
  - Python’s built-in `csv` module
  - Pandas library
- Extract and convert column data
- Understand why Pandas is the industry standard for tabular data
